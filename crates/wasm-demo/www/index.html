Nectar WASM Demo

Demonstrates parallel BMT (Binary Merkle Tree) hashing, chunk generation, and ECDSA postage stamping using wasm-bindgen-rayon. Rayon's parallel iterators work in the browser via SharedArrayBuffer and Web Workers. The stamping demo includes full ECDSA signing with random keypairs.

Note: This demo requires cross-origin isolation for SharedArrayBuffer. The server must send Cross-Origin-Opener-Policy: same-origin and Cross-Origin-Embedder-Policy: require-corp headers.
